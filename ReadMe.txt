1. Install Python
2. Install Tensorflow
3. Install Keras
4. Install h5py library
5. Install OpenCV

- mnist_training.py: Training model
- mnist_test.py:    Test trained model
- mnist_model.h5:   Model (will be loaded when testing)
- images: 10 images (0-9)

To run the mnist_test.py

python mnist_test.py image_name

E.g: python mnist_test.py images/0.jpg


You can also try to run mnist_training.py if you want to:
python mnist_training.py
